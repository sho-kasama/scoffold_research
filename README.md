# README

Railsでajaxを用いて簡単なインクリメンタルサーチを実装した


```
文字が入力される
↓
jsで読み取る
↓
コントローラーにjson形式でajax通信でデータを渡す
↓
そのデータを元に、like句を使って検索する
↓
検索結果をjsに返す
↓
受け取ったデータをappendで表示する
```
